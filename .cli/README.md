# feedsnap — Agent Discovery Reference

> ACLI v0.1.0 · Generated by `feedsnap skill`

feedsnap turns any RSS or Atom feed into a clean markdown or JSON digest.
It can also process multiple feeds at once via an OPML subscriptions file,
filter entries by date, and deduplicate entries across runs using SQLite.

## Quick start

```bash
pip install feedsnap

# Read a feed
feedsnap https://lobste.rs/rss

# Only new entries since last run
feedsnap https://lobste.rs/rss --dedup

# Multiple feeds, last 24 hours
feedsnap --opml feeds.opml --since 1d
```

## Discovery commands

```bash
feedsnap introspect          # Full command tree as JSON (ACLI format)
feedsnap skill               # This SKILL.md (agentskills.io format)
feedsnap --help              # Human-readable usage
feedsnap --version           # Installed version
```
